## 3. Corrections Made

### 3.1 JankyBorders Configuration File Syntax

**Location:** Section 4.2 and Section 8.4

**Issue:** The document showed a simple multi-line command format which, while functional, differs from the recommended array-based approach in official documentation.

**Before:**
```bash
borders \
    style=round \
    width=4.0 \
    ...
```

**After:**
```bash
#!/bin/bash

options=(
    style=round
    width=4.0
    ...
)

borders "${options[@]}"
```
