## Installation Notes

To activate these configurations:

1. **Symlink the configs** (if not already done):
   ```bash
   ln -sf ~/workspace/app.configs/aerospace/aerospace.toml ~/.aerospace.toml
   ln -sf ~/workspace/app.configs/sketchybar ~/.config/sketchybar
   ln -sf ~/workspace/app.configs/borders/bordersrc ~/.config/borders/bordersrc
   ```

2. **Restart services:**
   ```bash
   # Restart SketchyBar
   brew services restart sketchybar

   # Restart borders
   brew services restart borders

   # Or reload AeroSpace (which starts both)
   aerospace reload-config
   ```

3. **Prerequisites:**
   - Install Hack Nerd Font: `brew install --cask font-hack-nerd-font`
   - Install dependencies: `brew install sketchybar borders nikitabobko/tap/aerospace`
