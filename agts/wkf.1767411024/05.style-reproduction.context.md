## Target Visual Elements (from screenshot)

- Focused workspace indicated by pink/magenta circular highlight behind number
- Center: App name in rounded pill with warm gray border
- Right: Individual rounded pills for each status item with colored borders
- All items have dark semi-transparent backgrounds (0xEB1e1e2e)
- Pink/magenta accent color (0xFFD7448A) for highlights and workspace bracket

### Workspace Highlight System
CriticalElement uses a clever two-layer approach:
1. First layer: Regular space numbers (space.1-space.10)
2. Pink highlight dot: Animated circle that moves behind focused number
3. Second layer: Overlay numbers (space_ex.1-space_ex.10) positioned on top
4. Animation: `sketchybar --animate circ 15` for smooth transitions
