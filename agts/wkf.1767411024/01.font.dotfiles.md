# Font Settings Analysis: neutonfoo/dotfiles Sketchybar Configuration

Source: https://github.com/neutonfoo/dotfiles

## Summary

The sketchybar configuration uses **JetBrainsMono Nerd Font** as the primary font across all UI elements. The configuration is split between desktop and laptop variants with slightly different font sizes optimized for each display type.

## Font Settings

### Primary Font

```bash
FONT_FACE="JetBrainsMono Nerd Font"
```

The Nerd Font variant is essential as it includes special icon glyphs used throughout the status bar.

### Font Format

Sketchybar uses the format: `"FontName:Weight:Size"`

### Desktop Configuration (sketchybarrc-desktop)

| Element | Font Configuration | Size | Weight |
|---------|-------------------|------|--------|
| Default Icon | `"$FONT_FACE:Medium:20.0"` | 20.0 | Medium |
| Default Label | `"$FONT_FACE:Bold:14.0"` | 14.0 | Bold |
| App Name Label | `"$FONT_FACE:Bold:16.0"` | 16.0 | Bold |
| Separator Icon | `"$FONT_FACE:Bold:23.0"` | 23.0 | Bold |

### Laptop Configuration (sketchybarrc-laptop)

| Element | Font Configuration | Size | Weight |
|---------|-------------------|------|--------|
| Default Icon | `"$FONT_FACE:Medium:15.0"` | 15.0 | Medium |
| Default Label | `"$FONT_FACE:Medium:12.0"` | 12.0 | Medium |
| Front App Label | `"$FONT_FACE:Bold:12.0"` | 12.0 | Bold |
| Spotify Icon | `"$FONT_FACE:Bold:20.0"` | 20.0 | Bold |
| Weather Moon Icon | `"$FONT_FACE:Bold:22.0"` | 22.0 | Bold |
| Weather Icon | `"$FONT_FACE:Bold:15.0"` | 15.0 | Bold |

## Configuration Code Snippets

### Desktop Default Settings

```bash
FONT_FACE="JetBrainsMono Nerd Font"

sketchybar --default \
  icon.font="$FONT_FACE:Medium:20.0" \
  icon.padding_left=5 \
  icon.padding_right=5 \
  label.font="$FONT_FACE:Bold:14.0" \
  label.y_offset=1
```

### Laptop Default Settings

```bash
FONT_FACE="JetBrainsMono Nerd Font"

sketchybar --default \
  icon.font="$FONT_FACE:Medium:15.0" \
  label.font="$FONT_FACE:Medium:12.0"
```

### App-Specific Overrides (Laptop)

```bash
# Front app with bold label
sketchybar --set front_app \
  label.font="$FONT_FACE:Bold:12.0"

# Spotify with larger bold icon
sketchybar --set spotify \
  icon.font="$FONT_FACE:Bold:20.0"

# Weather with custom icon sizes
sketchybar --set weather \
  icon.font="$FONT_FACE:Bold:15.0"

sketchybar --set weather_moon \
  icon.font="$FONT_FACE:Bold:22.0"
```

## How to Reproduce These Settings

### 1. Install JetBrainsMono Nerd Font

```bash
# Using Homebrew
brew tap homebrew/cask-fonts
brew install --cask font-jetbrains-mono-nerd-font
```

Or download from: https://www.nerdfonts.com/font-downloads

### 2. Configure sketchybarrc

Create or edit `~/.config/sketchybar/sketchybarrc`:

```bash
#!/bin/bash

FONT_FACE="JetBrainsMono Nerd Font"

# Set default font settings
sketchybar --default \
  icon.font="$FONT_FACE:Medium:15.0" \
  icon.padding_left=5 \
  icon.padding_right=5 \
  label.font="$FONT_FACE:Bold:14.0" \
  label.padding_left=5 \
  label.padding_right=5
```

### 3. Reload Sketchybar

```bash
sketchybar --reload
```

## Font Weight Options

Sketchybar supports standard font weights:
- Regular
- Medium
- Bold
- Light
- Thin

The neutonfoo configuration primarily uses **Medium** for icons and **Bold** for labels to create visual hierarchy.

## Design Notes

- Desktop uses larger fonts (14-23pt) optimized for external monitors
- Laptop uses smaller fonts (12-22pt) for built-in Retina displays
- Nerd Font glyphs are used for icons throughout the configuration
- Bold weight is used for labels to improve readability
- Medium weight for icons provides a balanced visual weight
