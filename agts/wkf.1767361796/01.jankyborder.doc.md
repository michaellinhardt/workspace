# JankyBorders Documentation

A lightweight window border system for macOS 14.0+ by FelixKratz.

## 1. Main Features and Capabilities

- **Colored Window Borders**: Adds customizable borders around user windows on macOS
- **Performance Focused**: Does not rely on the accessibility API by default, making it faster than comparable tools
- **Lightweight**: Written in 99.4% C with minimal dependencies
- **Live Configuration**: Running a new instance with updated options modifies an already-running process without restart
- **Duplicate Prevention**: Built-in detection of already running process prevents multiple instances
- **HiDPI Support**: Retina resolution rendering available

## 2. How It Works

### Border Styles

Three border style options available:
- `round` - Rounded corners on borders
- `square` - Sharp square corners
- `uniform` - Uniform appearance

### Colors

Colors use hexadecimal format: `0xAARRGGBB`
- `AA` - Alpha channel (transparency)
- `RR` - Red channel
- `GG` - Green channel
- `BB` - Blue channel

Example: `0xff00ff00` creates a fully opaque green border.

### Active/Inactive States

- **active_color**: Border color for the currently focused window
- **inactive_color**: Border color for all unfocused windows
- **background_color**: Fill color for window backgrounds

## 3. Configuration Options

### Core Settings

| Option | Description | Example |
|--------|-------------|---------|
| `style` | Border style | `style=round` |
| `width` | Border thickness in points | `width=5.0` |
| `active_color` | Focused window color | `active_color=0xffe1e3e4` |
| `inactive_color` | Unfocused window color | `inactive_color=0xff494d64` |
| `background_color` | Window fill color | `background_color=0x00000000` |
| `hidpi` | Retina resolution rendering | `hidpi=on` |

### Focus Detection

| Option | Description |
|--------|-------------|
| `ax_focus` | Uses accessibility API for focus detection (`on`/`off`). Activates automatically if process has accessibility permissions. |

### Application Filtering

| Option | Description | Example |
|--------|-------------|---------|
| `blacklist` | Excludes specified apps | `blacklist="Safari,kitty"` |
| `whitelist` | Restricts borders to listed apps only | `whitelist="Code,Terminal"` |

### Configuration File

Location: `~/.config/borders/bordersrc`

When launched without arguments, JankyBorders automatically executes this file.

Example bordersrc:
```bash
borders style=round \
        width=6.0 \
        hidpi=on \
        active_color=0xffe1e3e4 \
        inactive_color=0xff494d64
```

## 4. AeroSpace Integration

JankyBorders integrates seamlessly with AeroSpace tiling window manager.

### Configuration in aerospace.toml

Use `after-startup-command` to launch JankyBorders with AeroSpace:

```toml
after-startup-command = [
    'exec-and-forget borders active_color=0xffe1e3e4 inactive_color=0xff494d64 width=5.0'
]
```

### Key Integration Features

- **Auto-start**: Borders launches automatically when AeroSpace starts
- **Duplicate Prevention**: JankyBorders detects if already running, preventing duplicate instances on AeroSpace restart
- **exec-and-forget**: Command runs in background without blocking AeroSpace startup

### Recommended Settings for Tiling WM

For tiling window managers like AeroSpace, consider:
- Higher contrast colors to distinguish focused window in tiled layout
- Moderate width (4.0-6.0) for visibility without excessive screen usage
- Round style often pairs well with modern macOS aesthetics

## 5. Appearance Customization

### Border Width

Set with `width=<float>` parameter:
- `width=3.0` - Subtle border
- `width=5.0` - Standard visibility
- `width=8.0` - Prominent border

### Special Color Effects

**Gradients:**
```bash
# Diagonal gradient from top-left to bottom-right
active_color="gradient(top_left=0xffff0000,bottom_right=0xff0000ff)"

# Alternative diagonal from top-right to bottom-left
active_color="gradient(top_right=0xffff0000,bottom_left=0xff0000ff)"
```

**Glow Effect:**
```bash
active_color="glow(0xff00ff00)"
```

### HiDPI/Retina Support

Enable with `hidpi=on` for crisp borders on Retina displays.

## 6. Installation

### Via Homebrew (Recommended)

```bash

brew install borders
```

### Running as Service

```bash
brew services start borders
```
